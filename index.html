phpredis binaries

Install instructions

Put both php_redis_*.dll and php_igbinary_*.dll in your php extensions folder and add both to php.ini.

Compile flags

--enable-redis=shared --enable-redis-session --enable-igbinary=shared --enable-redis-igbinary
